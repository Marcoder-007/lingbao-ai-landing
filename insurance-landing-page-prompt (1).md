# 保险销售AI助理 — Landing Page 开发 Prompt

## 项目概述

构建一个单页面（SPA）产品官网，推广"保险销售AI助理"产品。面向保险行业 Top Sales 和代理团队负责人。

**技术栈：** Next.js 14 + Tailwind CSS + Framer Motion（动画）
**部署：** Vercel
**语言：** 中文为主
**响应式：** 移动端优先，兼容桌面端

---

## 设计风格

### 整体调性
- **精英化 + 科技感**：不是廉价的 SaaS 页面，要有金融级的质感
- **浅色/白色主题**，搭配深蓝+金色点缀（传递信任、专业、稳重感）
- 参考风格：Apple 官网的简洁留白 + Stripe 的精致排版 + 高端私人银行官网的质感
- **不要**花哨的渐变、彩虹色、卡通插画
- 大面积留白，呼吸感强，内容区域居中收窄（max-width: 1200px）

### 配色方案
```
主背景：#FFFFFF（纯白）
次背景：#F7F8FA（浅灰，区块交替）/ #F0F4F8（淡蓝灰）
卡片背景：#FFFFFF（白色，用阴影区分层次）
主强调色：#1B3A6B（深海蓝）— 用于标题、重要文字
次强调色：#D4A853（琥珀金）— 用于 CTA 按钮、重点数字、图标高亮
辅助色：#2E8B57（沉稳绿）— 用于"收益""增长""成功"标记
文字主色：#1A1A2E（深色，接近黑）
文字次色：#6B7280（中灰，说明文字）
边框/分割：#E5E7EB（浅灰边框）
危险/痛点：#DC2626（红色，痛点部分用）
成功/解决：#16A34A（绿色，解决方案标记）
卡片阴影：0 2px 12px rgba(0,0,0,0.06)（轻柔阴影）
```

### 字体
- 标题：思源黑体 / Noto Sans SC（Bold/Black）
- 正文：Inter / 思源黑体（Regular）
- 数据/数字：JetBrains Mono 或 Space Grotesk（等宽，科技感）
- 英文辅助：Inter

### 动画
- 使用 Framer Motion
- 滚动触发渐入（fade-in-up），不要过度
- 数字用 countUp 动画（从 0 滚动到目标数字）
- Hero 区可以有轻微的几何图形/网格线装饰（极低透明度，subtle）
- 卡片 hover 有轻柔阴影加深效果 + 微微上浮（translateY -2px）

## 技术要求

### 性能
- Lighthouse 分数 > 90
- 首屏加载 < 2秒
- 图片使用 next/image + WebP
- 字体使用 next/font 预加载

### SEO
- 页面标题: "保险销售AI助理 — 30秒生成专业方案 | 智能保险销售工具"
- Meta description: "AI驱动的保险销售助理，30秒自动生成86列现金流方案书。覆盖500+保险产品，服务效率提升600倍。免费试用7天。"
- OG Image: 1200×630，深色背景+金色标题

### 动画细节
- Hero 数字: countUp 动画（2秒，easeOut）
- Section 进入: fadeInUp（stagger 0.1s）
- 卡片 hover: 阴影加深 + 微微上浮（translateY -2px, box-shadow 加深）
- CTA 按钮: hover 时微微放大（scale 1.02）+ 光泽滑过效果（shimmer）
- 定价卡片"专业版": 金色顶部边框 + 轻微阴影放大
- 背景装饰: 浅色几何图形/圆点网格（极低透明度，opacity 0.03-0.05）

### 组件库建议
- Headless UI 或 Radix UI（FAQ 手风琴）
- Framer Motion（动画）
- react-countup（数字动画）
- Lucide Icons（线性图标）

## 注意事项

1. **不要用任何 stock photo**，宁可用抽象图形/UI mockup
2. **所有文案已提供**，不需要用 lorem ipsum
3. **移动端优先**：保险销售员大部分用手机看
4. **浅色/白色主题为主**，保证清爽干净，深蓝+金色做点缀
5. **金色不要过度**，只用在 CTA 按钮、重点数字、图标高亮上；深蓝用于标题和重要文字
6. **中文排版**：行高 1.8，段落间距充足
7. 页面底部可以加一个微信二维码浮窗（固定右下角）
